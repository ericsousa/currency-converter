# 💱 Currency Converter CLI

A simple and functional currency converter developed in **TypeScript** that runs directly in the terminal. This project allows users to convert values between Brazilian Real (BRL) and US Dollar (USD) using manual rates or real-time quotes.

## 🚀 Features

- ✅ BRL to USD conversion.
- ✅ USD to BRL conversion.
- ✅ Manual exchange rate adjustment.
- ✅ **API Integration**: Automatic real-time quote fetching via [AwesomeAPI](https://docs.awesomeapi.com.br/).
- ✅ User-friendly Command Line Interface (CLI).

## 🛠️ Technologies Used

- **TypeScript**: Main language for static typing and robustness.
- **Node.js**: Execution environment.
- **prompt-sync**: For synchronous user interaction in the terminal.
- **Fetch API**: For consuming external data asynchronously.
